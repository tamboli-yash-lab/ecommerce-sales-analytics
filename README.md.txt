# E-Commerce Sales Analytics Dashboard

## Project Overview
This project analyzes e-commerce sales data using Python, SQL, PostgreSQL, and Power BI to generate business insights and interactive dashboards.

## Tools Used
- Python (Pandas)
- PostgreSQL
- SQL
- Power BI

## Dashboard Features
- Total Revenue KPI
- Total Profit KPI
- Monthly Sales Trend
- Regional Sales Performance
- Category Profitability
- Top 5 Products by Sales
- Interactive Filters

## Key Business Insights
- West region generated highest revenue.
- Electronics category produced highest profit.
- Camera was the highest-selling and most profitable product.
- February showed the weakest sales trend.

- Python data cleaning script
